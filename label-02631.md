### 生成内容

#### 1. 技术架构描述

本项目采用纯 Python 技术栈构建，以 PyTorch 深度学习框架为核心引擎，基于 torchvision 提供的 VGG16 预训练模型进行迁移学习微调。代码分层结构遵循 ML Pipeline 标准范式：数据层（prepare_data.py 负责数据集发现、合成生成、验证集→测试集划分）、训练层（train.py 封装模型构建、优化器/调度器配置、训练循环与早停逻辑）、评估层（evaluate.py 实现指标计算、混淆矩阵动态分析、Grad-CAM 可视化）、调优层（retrain.py 管理多策略自动调参）、编排层（run_pipeline.py 串联全流程并集成环境预检）。数据管理采用文件系统目录结构（ImageFolder 约定：dataset/{train,val,test}/{class_name}/），配合 classes.json 作为类别元数据的单一数据源。核心依赖库包括 scikit-learn（分类指标计算）、matplotlib + seaborn（可视化输出）、Pillow（图像 I/O 与合成数据生成）、numpy（数值计算），构建工具使用 pip + requirements.txt 管理依赖，Docker 多阶段构建实现跨平台（ARM/X86）容器化部署。

#### 2. 数据与交互描述

数据持久化完全基于文件系统：训练数据以 JPEG 图片形式存储在 dataset/ 目录树中，模型权重保存为 PyTorch .pth checkpoint 文件（含 model_state_dict、optimizer_state_dict、epoch、config、class_to_idx 完整快照），评估指标输出为 JSON（test_metrics.json）和纯文本（confusion_analysis.txt、dataset_summary.txt），可视化图表保存为 PNG（confusion_matrix.png、per_class_metrics.png、training_curves.png、gradcam/*.png）。项目不涉及敏感信息处理（无用户数据、无认证凭据），所有配置通过 argparse 命令行参数传入（--epochs、--lr、--tag 等），支持 JSON 配置文件回读（config.json）。交互模式为纯 CLI 批处理：用户通过 `python run_pipeline.py` 一键触发全流程，或分步执行各脚本；日志系统采用 RotatingFileHandler 轮转机制（10MB/文件，保留 3 份历史），同时输出到控制台和日志文件，确保长时间训练过程可追溯。无 GUI/Web 界面，无快捷键/动画/事件触发等 UI 交互。

#### 3. 项目定位描述

VGG16 Animal Image Classifier 是一个基于 PyTorch + VGG16 迁移学习的 4 类动物（cat、dog、tiger、lion）图像分类训练与评估流水线，面向机器学习工程师和研究人员，提供从数据准备、模型训练、多维度评估（准确率/召回率/F1/混淆矩阵动态分析/Grad-CAM 可视化）到自动调优的端到端解决方案。

#### 4. 界面设计描述

本项目的视觉输出采用 matplotlib + seaborn 科学绘图风格，配色方案遵循信息可视化最佳实践：混淆矩阵使用 Blues（原始计数）和 Oranges（归一化百分比）双色系 colormap，通过冷暖色对比区分绝对值与相对比例；每类指标对比柱状图采用三色方案——Precision 使用 #4CAF50（Material Green）、Recall 使用 #2196F3（Material Blue）、F1 Score 使用 #FF9800（Material Orange），色彩饱和度适中且色盲友好；训练曲线图使用蓝/橙双色区分 train/val 趋势。布局结构上，混淆矩阵采用 1×2 并排子图（16×6 英寸），每类指标采用分组柱状图（10×6 英寸），Grad-CAM 热力图采用 1×3 并排展示（原图 + 热力图 + 叠加图，12×4 英寸），所有图表统一使用 150 DPI 输出、tight_layout 自动调整边距。视觉层次通过 seaborn heatmap 的 annot 数值标注、柱状图顶部数值标签、0.3 透明度网格线实现数据可读性分层，整体呈现专业学术论文级别的可视化质量。

- 从 `classes.json` 读取白名单类别

- 即使 `dataset/train/elephant/` 存在，也不会被加载到训练集中
- 日志记录发现的白名单外目录

```python
class FilteredImageFolder(datasets.ImageFolder):
    def find_classes(self, directory):
        classes_list, class_to_idx = super().find_classes(directory)
        filtered = [c for c in classes_list if c in allowed]
        new_idx = {c: i for i, c in enumerate(filtered)}
        return filtered, new_idx
```
